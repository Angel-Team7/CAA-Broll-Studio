# Edenrise B-roll Cockpit

Static review dashboard for approving license-cleared B-roll per lesson.

- Open `index.html` (via GitHub Pages) — pick a lesson, hover to preview, click to approve.
- **Save approvals** / **Send to editor** write `selections/<lesson>.json`.
- **🔁 Needs different** flags a scene for re-shoot (with a reason) → recorded in the same JSON.

Full 4K footage stays on the SSD; this repo holds only thumbnails + 480p previews + JSON.
